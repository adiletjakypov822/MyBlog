## General Info

* This is Web Application, where you can follow me.

## API Description

* To view all blogs you should access "/"
* To add book you should access "/blog/add"
* To update book you should access "/blog/{id}/update"
* To delete book you should access "/blog/{id}/delete"
* To read blog details you should access "/blog/${id}"
* To view about page ypu should "/about"
